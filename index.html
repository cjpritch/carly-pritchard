The Whiskey Cabinet


Made with HTML, CSS, Bulma, Javascript, MySQL, Express.js, Handlebars, Node.js, and Bcrypt.

The Whiskey Cabinet is a simple, easy to use application that helps connect whiskey enthusiasts. When entering the website the user will be prompted to either login in, or create an account. After being logged in the user can browse all of the previously reviewed whiskeys. New reviews can also be added, and will then be available to review.

Trivia Trainer


Made with HTML, CSS, Materialize, and Javascript.

Trivia Trainer is an application that generates randomized trivia questions for the user based on their selections. The user has a choice of the number of trivia questions, category types, or even to randomly generate questions from every category. When the “start training” button is pressed, the questions associated with the user selections are displayed, along with a button to reveal the answer.
